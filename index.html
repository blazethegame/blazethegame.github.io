BLAZE

Blaze, is a free physics based, action-packed multiplayer shooter game. There is an endless amount of content, from campaign to various multiplayer game modes, including competitive mode. Furthermore we builded an easy to use level editor, so that the players can build maps and new game modes. The game will never be pay-to-win nor paying money will make anyone advantageous as we believe it kills the competitiveness in games.
